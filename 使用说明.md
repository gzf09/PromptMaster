# PromptMaster 使用说明

## 登录与注册

打开 http://47.107.238.118 ，输入用户名和密码登录。也可以点击「访客浏览」查看社区公开的 Prompt。

## 核心概念

### 分类 = 大场景

分类用于划分 **大的应用场景**，比如系统内置的：

- 编程 — 代码生成、调试、重构类 Prompt
- 写作 — 文案、博客、营销内容
- 图像生成 — Midjourney、DALL-E、Stable Diffusion 等
- 数据分析 — 数据处理、报表、可视化
- 学习 — 知识问答、语言学习、考试备考
- 产品规划 — 产品需求、PRD、规划文档
- 架构设计 — 系统架构、技术方案
- PPT制作 — 演示文稿、汇报材料

你也可以创建自定义分类（侧边栏「分类」旁的 `+` 按钮），并选择一个图标。

### 标签 = 灵活标记

标签用于 **细粒度的交叉分类**，一个 Prompt 可以打多个标签。推荐用标签来区分：

- **应用/工具**：`chatgpt`、`claude`、`midjourney`、`cursor`、`copilot`
- **语言**：`中文`、`english`、`日本語`
- **技术栈**：`react`、`python`、`golang`、`sql`、`docker`
- **用途**：`模板`、`调试`、`优化`、`翻译`、`总结`

这样你可以通过热门标签快速筛选，比如点击 `#react` 立刻找到所有 React 相关的 Prompt，不管它在「编程」还是「学习」分类下。

### 组合使用示例

| Prompt | 分类 | 标签 |
|--------|------|------|
| React 组件生成器 | 编程 | `react`, `typescript`, `chatgpt` |
| Python 数据清洗脚本 | 数据分析 | `python`, `pandas`, `claude` |
| Midjourney 人像提示词 | 图像生成 | `midjourney`, `portrait`, `english` |
| 英文邮件润色 | 写作 | `english`, `翻译`, `chatgpt` |
| Go 项目代码审查 | 编程 | `golang`, `claude`, `code-review` |

## 日常操作

**创建 Prompt**：点击右上角「新建 Prompt」，填写标题、内容、选择分类，添加标签，设置公开或私有。

**搜索**：顶部搜索栏支持按标题、内容、标签模糊搜索。可配合分类下拉框进一步筛选。

**热门标签**：页面上方会显示使用最多的标签，点击即可快速筛选/取消。

**收藏**：点击 Prompt 卡片上的星标收藏，在侧边栏「收藏」视图集中查看。

**复制**：点击卡片上的复制按钮，或打开详情后点击内容区域右上角的复制图标。

**Fork**：看到他人的公开 Prompt 想在此基础上修改？点击 Fork 图标，自动创建一份副本到自己名下，自由编辑。

**社区**：将 Prompt 设为「公开」后，所有用户都能在「社区」视图看到并复制使用。

**修改密码**：点击侧边栏底部的用户头像区域，在弹出菜单中选择「修改密码」。

## 权限说明

| 角色 | 能力 |
|------|------|
| 管理员 | 管理用户（添加/删除）、编辑和删除任意 Prompt、管理分类、控制注册开关 |
| 普通用户 | 管理自己的 Prompt、Fork 他人公开 Prompt、创建自定义分类 |
| 访客 | 仅浏览社区公开 Prompt，可复制内容 |

每张 Prompt 卡片都会显示创建者名称。

## 小技巧

- 标签输入支持多种分隔符：逗号 `,` `，`、顿号 `、`、分号 `;` `；`、回车，可以一次添加多个标签
- 保存 Prompt 时，输入框中未按回车的标签会自动提交，不会丢失
- 给常用 Prompt 加上统一前缀标签（如 `daily`），方便每日快速定位
- 私有 Prompt 只有自己能看到，适合存放包含敏感信息的 Prompt
- 支持中英文界面切换和亮色/暗色主题切换（侧边栏底部）
